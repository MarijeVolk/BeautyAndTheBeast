Repository for Beauty and the Beast video game project.

To run the most recently exported version of the desktop game, double click Game.jar (Note that Game.jar may not work if it is moved to a different location).

Beauty:				Core project code

Beauty-desktop:		Wrapper project to convert to a desktop application

Beauty-android:		Wrapper project to convert to an android application

Beauty-html:		Wrapper project to convert to a web application

libgdx:				example libgdx projects
